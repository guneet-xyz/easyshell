### Conclusion ###

1. .Mounts is empty array []  — no host filesystem is mounted into the container.
2. .HostConfig.Binds is null — no bind mounts configured.
3. 'which docker' returns nothing (docker CLI not installed).
4. 'docker --version' fails with 'docker: not found'.
5. /var/run/docker.sock does NOT exist in the container.

The submission-manager container has zero ability to talk to the Docker daemon.
It can only reach session-manager via HTTP, as designed.
